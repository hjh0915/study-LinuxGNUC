链接器脚本
=========
SECTIONS命令
------------
包括：ENTRY命令、符号赋值命令、输出节的描述、overlay的描述

SECTIONS {
    . = 0x10000;
    .text : { * (.text)}
    . = 0x8000000;
    .data : { * (.data)}
    .bss : { * (.bss)}
}

MEMORY命令
----------
内存区域/内存块的定义

MEMORY {
    name [(attr)] : ORIGIN = origin, Length = len
    ...
}

区分链接器脚本和makefile
======================
链接器脚本：主要用于控制链接过程的布局过程，这指定输出目标的节所处位置、由哪些输入节构成。 
makefile：帮助大型程序的编译自动化，分布在不同目录的众多源代码和需要生成的目标文件，使用makefile依赖关系
